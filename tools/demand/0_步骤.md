# 找到 vercel 给的 CNAME 和 A 记录：

在 vercel 首页上点击 Settings，再点击点 Domains，输入域名并 save(注意必须正确复制域名，不要拼写出错)，这时候会出来 2 个记录，一个是域名，一个是 www.域名，分别点开它们，截图它们的 DNS records(有 type name value proxy 等等)，准备去 Claudeflare 里面添加

# Claudeflare 里面添加 DNS 记录

https://dash.cloudflare.com/610b80b08aa02848902d5fbdafbc9790/home/domains在 Cloudflare 里面，点击此次域名，点击 DNS，点击Add Records,添加 A 记录，1）Type：A
2）Name：@ 3）Value：填 Vercel 给的 IP 4）DNS only（灰云），添加 CNAME 记录，1）Type：CNAME 2）Name：www 3）Value：填 Vercel 给的 IP 4）DNS only（灰云）。

注意：1）如果 @ 或 www 已经有别的 A/CNAME，删掉或改掉，不然它们会互相打架 2）如果有 AAAA（IPv6）记录，先删掉，Vercel 经常会因此判定异常
